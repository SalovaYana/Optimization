Detailed Description

Class representing a rectangular area in N-dimensional space.

This class encapsulates the definition of a rectangular area, defined by its lower and upper bounds in each dimension. It is intended for use in optimization problems, where such an area specifies the bounds within which the optimization is to be conducted.
